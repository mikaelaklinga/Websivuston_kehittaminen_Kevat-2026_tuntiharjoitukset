Tämä on kurssin viimeinen tehtävä: Projektityö verkkosivut
Tein verkkosivut oikealle yritykselle. 
Sivulla on neljä sivua: Etusivu, Palvelut, Tietoa, Ajanvaraus 

Sivusto on suunniteltu ja määritelty niin että tämän tehtäväpalautuksen jälkeen suoritamme domain hankinnat, ohjaamme "ota yhteyttä" lomakkeen toimminnon Formspreen kautta suoraan yhteydenoton sähköpostiin, ilman että shäköpostia tarvitsee uudelleen kirjoittaa ja verkkosivu julkaistaan myös virallisesti kaikille. 

Nyt olen tehnyt verkkosivun siihen pisteeseen valmiiksi kuin tämä tehtävänanto sitä edellyttää.
